# StudyGlade MVP

## Setup
1. `npm install`
2. Create `.env` from above and add MongoDB URI, JWT secret, Cloudinary keys
3. `npm run dev`

## Features
- Student: post questions, browse library, unlock documents
- Tutor: browse questions, accept assignments, upload documents
- Admin: approve tutors, view all users/questions/documents
- Wallet: virtual balance, deduct on post/unlock, add funds (simulated)

## Deployment on Render
- Connect GitHub repo
- Build command: `npm install`
- Start command: `npm start`
- Add env vars
